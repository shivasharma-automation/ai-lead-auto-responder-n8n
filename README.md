# 🤖 AI Lead Auto Responder (n8n)

An AI-powered customer support and lead management workflow built with n8n.

## 🚀 Features

- HTML Lead Capture Form
- Webhook Integration
- Google Sheets Database
- AI Response Generation
- Multi AI Provider Support (Groq, OpenAI, Claude)
- Gmail Auto Reply
- Client Branding
- Lead Status Tracking

## 🛠 Tech Stack

- n8n
- Google Sheets
- Gmail
- Groq API
- OpenAI API
- Claude API
- HTML
- JavaScript

## 📌 Workflow

Customer Form → Webhook → Google Sheets → AI Provider → Gmail → Update Google Sheets
